3.3. ソースの入手

Pgpool-IIのソースは、Webサイトhttp://www.pgpool.netのダウンロードページから入手することができます。 pgpool-II-3.6.4.tar.gzという名前のファイルを入手してください。 ファイルを入手したら、展開します。 

tar xf pgpool-II-3.6.4.tar.gz

これにより、カレントディレクトリ以下にpgpool-II-3.6.4というディレクトリが作成され、Pgpool-IIのソースが展開されます。 この後のインストール手順を行うために、このディレクトリに移動してください。
