pcp_recovery_node

名前

pcp_recovery_node --  Pgpool-II のノードのデータを再同期させた上で復帰させる

概要

pcp_recovery_node [options...] [node_id]

説明

pcp_node_info は Pgpool-II のノードのデータを再同期させた上で復帰させます。

オプション

-n node_id
--node-id=node_id

バックエンドノードのインデックスを指定します。

その他

pcp_common_options を参照してください。
