IV. リファレンス

ここには Pgpool-II のリファレンス情報があります。

このリファレンスは伝統的な "man" ページとしても入手できます。

目次
I. サーバコマンド
pgpool --  Pgpool-IIメインサーバ
II. PCP コマンド
pcp_common_options --  PCP コマンドには共通する引数
pcp_node_count --  Pgpool-II の pgpool.conf で定義されたノードの総数を表示する
pcp_node_info --  指定されたノードの情報を表示する
pcp_watchdog_info --  Pgpool-II の watchdog ステータスを表示します
pcp_proc_count --  Pgpool-II の子プロセスのプロセス ID を一覧表示する
pcp_proc_info --  Pgpool-II の子プロセス情報を表示する
pcp_pool_status --  pgpool.conf のパラメータ設定値を取得する
pcp_detach_node --  Pgpool-II のノードを切り離す
pcp_attach_node --  Pgpool-II のノードを復帰させる
pcp_promote_node --  Pgpool-II のノードをマスターに昇格させる
pcp_stop_pgpool --  Pgpool-II を指定されたモードでシャットダウンする
pcp_recovery_node --  Pgpool-II のノードのデータを再同期させた上で復帰させる
III. 他のコマンド
pg_md5 --  md5 暗号化したパスワードを生成する
pgpool_setup --  Pgpool-II クラスタのテスト環境を構築するツール
watchdog_setup --  watchdog 機能を有効にした Pgpool-II クラスタのテスト環境を構築するツール
IV. SQL 型コマンド
PGPOOL SHOW -- パラメータの設定値を表示する
PGPOOL SET -- 実行時パラメータの設定を変更する
PGPOOL RESET -- 設定パラメータの値をデフォルト値に戻す
SHOW POOL STATUS --  設定パラメータの名前と値、説明を表示する
SHOW POOL NODES --  DB ノードのリストを表示する
SHOW POOL_PROCESSES --  接続待ち、あるいは接続中 Pgpool-II の子プロセスの状態を表示する
SHOW POOL_POOLS --  Pgpool-II のコネクションプールの状態を表示する
SHOW POOL_VERSION --  Pgpool-II のバージョン情報を表示する
SHOW POOL_CACHE --  クエリキャッシュのヒット率や、キャッシュストレージの状況を表示する
V. pgpool_admエクステンション
pgpool_adm_pcp_node_info -- 指定ノードIDの情報を表示する関数
pgpool_adm_pcp_pool_status -- pgpool.confのパラメータを取得する関数
pgpool_adm_pcp_node_count -- ノード数を取得する関数
pgpool_adm_pcp_attach_node -- 指定ノードをアタッチする関数
pgpool_adm_pcp_detach_node -- 指定ノードをデタッチする関数
