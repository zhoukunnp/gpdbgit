pcp_proc_count

名前

pcp_proc_count --  Pgpool-II の子プロセスのプロセス ID を一覧表示する

概要

pcp_proc_count [options...]

説明

pcp_proc_count は Pgpool-II の子プロセスのプロセス ID を一覧表示します。複数ある場合は空白文字で区切られます。

オプション

pcp_common_options を参照してください。
